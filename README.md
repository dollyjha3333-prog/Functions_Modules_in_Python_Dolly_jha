## 📌 Assignment Tasks

### ✅ Task 1: Calculate Factorial Using a Function

### Objective

Write a Python program that:

- Defines a function to calculate the factorial of a number.
- Uses a loop to compute the factorial.
- Returns the calculated value.
- Accepts input from the user.
- Displays the factorial.

### Concepts Covered

- Functions
- Parameters
- Return Statement
- For Loop
- User Input

### Sample Output

```
Enter the number of which you want to find the factorial: 5

The Factorial of the number 5 is 120
```

---

### ✅ Task 2: Mathematical Calculations Using the Math Module

### Objective

Write a Python program that:

- Takes a number as input.
- Uses Python's `math` module to calculate:
  - Square Root
  - Natural Logarithm (Base e)
  - Sine (in radians)
- Displays all calculated values.

### Concepts Covered

- Importing Modules
- math.sqrt()
- math.log()
- math.sin()
- User Input

### Sample Output

```
Enter a number: 25

The Square root of the number 25 is 5.0
The Natural log of the number 25 is 3.2188758248682006
The Sine of the number 25 is -0.13235175009777303
```
